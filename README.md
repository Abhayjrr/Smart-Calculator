# Smart-Calculator
# 🧮 Smart Scientific Calculator

A modern **Scientific Calculator** built using **Python** and **Tkinter** with an intuitive graphical user interface. This calculator supports basic arithmetic, advanced scientific operations, memory functions, keyboard shortcuts, and calculation history.

---

## 📌 Features

### ✅ Basic Calculator
- Addition
- Subtraction
- Multiplication
- Division
- Decimal calculations
- Parentheses support
- Clear display
- Backspace/Delete

### 🧪 Scientific Functions
- Sine (Degrees)
- Cosine (Degrees)
- Tangent (Degrees)
- Square Root
- Logarithm (Base 10)
- Natural Log (ln)
- Power (xʸ)
- Factorial
- π (Pi)
- Euler's Number (e)

### 💾 Memory Functions
- M+ (Memory Add)
- M- (Memory Subtract)
- MR (Memory Recall)
- MC (Memory Clear)

### 📜 Calculation History
- Stores every calculation
- Auto-scroll to latest result
- Double-click any history item to reuse the expression
- Clear History button

### ⌨ Keyboard Support
| Key | Action |
|------|---------|
| Numbers | Input |
| + - * / | Operators |
| Enter | Calculate |
| Backspace | Delete last character |
| Esc | Clear display |
| ^ | Power operator |

### 🔒 Safe Expression Evaluation
The calculator evaluates expressions inside a restricted environment that only exposes approved mathematical functions. This prevents access to Python built-ins while still allowing scientific calculations.
